### LLD of TIC-TAC-TOE GAME

## Requirements
- Design a tic-tac-toe game of N*N board having two players
- provide a way to launch the game
- Players should be able to make moves 1 by 1
- End the game if there's a winner or there are no more moves left
- One of the 2 players can be a computer

## Future Scope
- Computer players can have different difficulty level
  - Easy
  - Medium
  - Hard
